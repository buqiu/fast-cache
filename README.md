# fast-cache
短小精悍的前端缓存工具，防止内存『侧漏』。

# 简介

前端缓存工具

# 安装下载

- [下载地址](https://github.com/buqiu/fast-cache/releases)
- 安装 `npm i buqiu-fast-cahce`
- [CND](http://unpkg.com/buqiu-fast-cache/release/bundle.js)

# 快速使用

```js
var FC = window.FastCache;
var cache = new FC();
cache.set('a', 100);
alert(cache.get('a'));
```
